** images文件夹使用说明
1.图片全部添加再images文件夹内
2.header文件夹中添加首页大图背景图片，尺寸参考header_2这张图
3.hotel是所有酒店图片，需要根据不同酒店建立文件夹
4.news是所有新闻图片，需要根据不同新闻建立文件夹
5.navbar是储存导航条背景图

** 如何使用data内的文件

2.以新闻数据为列
